## Code Changes

### Bug Fix
**File**: `src/ldpc/core.rs` (lines 846-858)
- Added message extraction from systematic codeword
- Decoder now correctly returns k-bit message instead of n-bit codeword
- Preserves systematic `[message | parity]` convention

```rust
// Extract message bits from systematic codeword [message | parity]
let k = self.code.k();
let mut message = BitVec::with_capacity(k);
for i in 0..k {
    message.push_bit(decoded_codeword.get(i));
}
```
